## Key framing

The EXPLORE Act is not a disc-golf-specific statute. It is a public-lands recreation law that gives BLM tools around inventories, accessibility, permitting, gateway communities, visitor data, and partnerships. Those tools fit disc golf unusually well, but the smartest framing is **"a low-impact day-use/accessibility/stewardship package that includes disc golf"** — not a claim that the statute expressly authorizes disc golf course construction.
